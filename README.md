#yandexmoney-simplacms

Модуль оплаты yandexmoney-simplacms необходим для интеграции с сервисом [Яндекс.Касса](http://kassa.yandex.ru/) на базе CMS SimplaCMS. 

Доступные платежные методы, если вы работаете как юридическое лицо:
* **Банковские карты** -  Visa (включая Electron), MasterCard и Maestro любого банка мира
* **Электронные деньги** - Яндекс.Деньги и WebMoney
* **Наличные** - [Более 170 тысяч пунктов](https://money.yandex.ru/pay/doc.xml?id=526209) оплаты по России
* **Баланс телефона** - Билайн, МегаФон и МТС
* **Интернет банкинг** - Альфа-Клик, Сбербанк Онлайн, Промсвязьбанк и MasterPass

###Установка модуля
Для установки данного модуля необходимо поместить файлы из [архива](https://github.com/yandex-money/yandex-money-cms-simplacms/archive/master.zip) в папку `/payment/YandexMoney` с заменой на новые.

Пожалуйста, обязательно делайте бекапы!
